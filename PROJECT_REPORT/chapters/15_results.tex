\section{Performance Evaluation Overview}

We evaluated NOMA-GNN on a single 500-user deployment scenario, comparing it against four baseline methods: Static Pairing, Balanced Pairing, Bipartite Matching, and Blossom Matching. The evaluation focused on two primary metrics: system throughput and computational complexity.

\end{itemize}

\section{Summary of Key Results}

Based on testing on a single 500-user deployment scenario:

\begin{enumerate}
    \item \textbf{Throughput:} 96.5\% of optimal (70.74 Gbps vs 73.30 Gbps)
    \item \textbf{Speedup:} 52.9× faster than Blossom (846 ms vs 44.7 s)
    \item \textbf{Complexity:} $O(N \log N)$ vs $O(N^3)$ for optimal methods
    \item \textbf{Pairing Efficiency:} 90\% (225 pairs formed out of 250 possible)
    \item \textbf{Scalability:} Successfully handles high-density scenario (N=500)
    \item \textbf{Sum-Rate:} 15.72 bits/s/Hz (99.5\% of optimal 15.80 bits/s/Hz)
\end{enumerate}

\section{Limitations and Future Work}

\subsection{Current Limitations}

\begin{itemize}
    \item \textbf{Single Scenario Testing:} Evaluation based on one 500-user deployment. Statistical validation across multiple scenarios needed.
    
    \item \textbf{Fixed Density:} Training and testing only on N=500. Generalization to variable densities (N ∈ \{50, 100, 200, 1000\}) not evaluated.
    
    \item \textbf{No Classification Metrics:} AUC, precision, recall not computed (requires labeled test pairs).
    
    \item \textbf{No Regression Metrics:} MAE, RMSE, R² for sum-rate and power allocation not evaluated.
    
    \item \textbf{Fairness Analysis Missing:} Jain's fairness index and per-user rate distributions not computed.
    
    \item \textbf{Channel Model Limited:} Only tested on 3GPP UMa. Performance under other models (UMi, InH) unknown.
    
    \item \textbf{Ablation Studies Not Performed:} Impact of multi-task learning, feature importance, network depth, and sampling strategies not investigated.
\end{itemize}

\subsection{Future Work}

\begin{itemize}
    \item \textbf{Multi-Scenario Evaluation:} Test on diverse deployments for statistical validation
    
    \item \textbf{Variable Density:} Train on mixed densities (N ∈ \{100, 200, 500, 1000\})
    
    \item \textbf{Comprehensive Metrics:} Compute classification and regression performance
    
    \item \textbf{Fairness Analysis:} Evaluate user-level throughput distribution
    
    \item \textbf{Ablation Studies:} Investigate impact of architecture choices
    
    \item \textbf{Generalization Testing:} Evaluate on unseen densities, channel conditions, and cell geometries
    
    \item \textbf{Real-World Deployment:} Field testing in actual base station environment
\end{itemize}

\section{Conclusion}

These results demonstrate that NOMA-GNN achieves a compelling trade-off between optimality (96.5\%) and computational efficiency (52.9× speedup), validating the feasibility of GNN-based resource allocation for high-density 5G/6G networks. While tested on a single scenario, the approach shows that near-optimal performance with polynomial complexity is achievable, addressing the computational bottleneck of traditional optimal methods.
