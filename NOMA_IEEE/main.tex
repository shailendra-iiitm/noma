\begin{abstract}
Power-Domain Non-Orthogonal Multiple Access (PD-NOMA) is a promising technology for 5G and beyond networks, enabling spectral efficiency improvements through intelligent user pairing and power allocation. However, traditional optimization-based pairing strategies suffer from high computational complexity ($O(N^3)$), limiting real-time deployment. This paper proposes NOMA-GNN, a novel Graph Neural Network framework based on GraphSAGE architecture that reformulates user pairing as a graph representation learning problem. We generate realistic Channel State Information using the 3GPP TR 38.901 Urban Macro model and develop a multi-task learning approach that jointly predicts pairing decisions, sum-rates, and power allocations while enforcing physical constraints including Successive Interference Cancellation (SIC) feasibility and angular separation. Comprehensive experiments on 50,000 scenarios demonstrate that NOMA-GNN achieves 95.8\% of optimal throughput with 92.5\% AUC for pairing classification, outperforming Static (81.4\%), Balanced (88.0\%), and Bipartite (93.4\%) baselines. The proposed method reduces inference complexity to $O(N \log N)$ with 20$\times$ speedup over Blossom matching, enabling millisecond-latency scheduling suitable for practical 5G New Radio deployments. Ablation studies validate the importance of multi-task learning, hard negative sampling, and angular features for achieving robust generalization across varying user densities and channel conditions.
\end{abstract}
