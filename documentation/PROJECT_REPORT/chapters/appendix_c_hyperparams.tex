\section{Summary}

The selected hyperparameters balance:
\begin{itemize}
    \item \textbf{Performance:} 96.5\% of optimal throughput on 500-user scenario
    \item \textbf{Efficiency:} 846 ms inference time for N=500 users
    \item \textbf{Scalability:} 128K parameters (deployable on edge devices)
    \item \textbf{Training:} Converges in ~120 epochs on 200 training scenarios
\end{itemize}

All configurations are stored in \texttt{config.py} for reproducibility.
